# 📱 OrgTrack

**OrgTrack** is a mobile application built with **Flutter** to help manage student organizations efficiently.
The app simplifies coordination by providing tools for **event management, attendance tracking, notifications, and member administration** in one integrated platform.

This project was developed as a **collaborative academic project** aimed at improving organization workflow, transparency, and communication among members.

---

# ✨ Features

## 🔐 Authentication

Secure user authentication system.

* User registration & login
* Role-based access (Admin / Member)
* Session management

---

## 📅 Organization Agenda Management

Manage organizational activities and schedules.

* Create agenda
* Edit agenda
* Delete agenda
* Active & expired agenda tracking

---

## ✅ Attendance System

Attendance tracking system for organizational activities.

Features include:

* Attendance based on agenda
* Admin lock / unlock attendance
* Attendance recap
* Attendance history for expired agendas

---

## 🔔 Notifications

Stay updated with organization announcements and activities.

* Event notifications
* Organization announcements
* Push notification support

---

## 👤 Profile & Member Management

Manage member data and access roles.

Features:

* Member profile
* Role management
* Organization member data

---

## 🌙 Theme Support

* Dark Mode
* Light Mode

Provides a better user experience across different environments.

```

```

---

```

```

---

```
flutter run
```
